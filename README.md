# Forms Briefing - AI Agent

Este projeto é um formulário interativo estilo "wizard" para briefing de Agentes de IA, desenvolvido para a OI Digital Media.

## Tecnologias

- React
- TypeScript
- TailwindCSS
- Vite
- Framer Motion
- Zustand

## Como rodar

1. Clone o repositório
2. Instale as dependências:
   ```bash
   npm install
   ```
3. Rode o servidor de desenvolvimento:
   ```bash
   npm run dev
   ```

## Deploy

O projeto está configurado para deploy em servidores Apache (como SiteGround) com suporte a roteamento SPA via `.htaccess`.

Para gerar a build de produção:
```bash
npm run build
```
Os arquivos estarão na pasta `dist`.
